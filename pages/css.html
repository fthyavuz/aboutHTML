Property Describe
CSS Color Colors are specified using predefined color names, or RGB, HEX, HSL, RGBA, HSLA values.
CSS Backgrounds The CSS background properties are used to add background effects for elements.
CSS Borders The CSS border properties allow you to specify the style, width, and color of an element's border.
CSS Margins Margins are used to create space around elements, outside of any defined borders.
CSS Paddings Padding is used to create space around an element's content, inside of any defined borders.
